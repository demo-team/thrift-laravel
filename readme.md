## Laravel with Thrift

Laravel 与thrift 结合开发。使用到artisan。

### 服务器端
```
shell> php artisan thrift:demo:server 

```

服务器端会要求选择 单进程和多进程模式 
1: 单进程模式
2: 多进程模式



### 客户端
```
shell> php artisan thrift:demo:client

```

### 

`注意`：运行本项目之前，需要执行 composer install
